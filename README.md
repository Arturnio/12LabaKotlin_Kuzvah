# Лабараторная работа # 9-10: Продвинутое ООП на Котлине
##  Описание
Лабараторная работа посвещена изучению продвинутых концепций объектно-ориентированного программирования на котлин
## Структура проекта
Проект содержит примеры реализации продвинутых механизмов ООП а также пояснения к ключевым концепциям
## Как запустить проект 
1. клонируйте репозиторий
```bash
git clone <URL_репозиторий>
```
2. Откройте проекет в Intelij IDEA.
3. Запустите любой пример через контекстное меню или напрямую из main
